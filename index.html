Tienda De Mascotas

Esta aplicación de comercio electrónico permite a los usuarios explorar y comprar una variedad de productos de manera fácil y atractiva. Incluye características como la gestión del carrito de compras, la visualización de detalles del producto y un proceso de checkout intuitivo. La aplicación está construida con React y Firebase, asegurando una experiencia de usuario fluida y actualizaciones en tiempo real. Además, su diseño responsivo garantiza un uso óptimo tanto en dispositivos móviles como en pantallas de escritorio.

Tienda de ropas

Este proyecto desarrolla el backend para una tienda de ropa en línea utilizando MongoDB base de datos NoSQL, JavaScript, Express.js y Handlebars. La API permite gestionar productos, usuarios y pedidos. El backend sigue el patrón MVC, utilizando Express.js para las rutas y controladores, Mongoose para los modelos en MongoDB, Handlebars para las vistas y middleware para autenticación, logging y manejo de errores.
